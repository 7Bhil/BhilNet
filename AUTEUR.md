### 👨‍💻 À Propos

Je suis un développeur passionné par la création d'outils de communication innovants. BhilNet est mon projet personnel visant à créer une solution de messagerie locale simple, sécurisée et efficace.

### 🎯 Objectifs du Projet

- **Simplicité** : Une interface intuitive pour tous les utilisateurs
- **Sécurité** : Chiffrement des messages pour protéger la vie privée
- **Performance** : Communication rapide et efficace sur réseau local
- **Accessibilité** : Compatible avec différentes plateformes

### 🚀 Compétences

- **Python** : Langage principal pour ce projet
- **Réseau** : TCP/UDP, sockets, communication client-serveur
- **Sécurité** : Chiffrement AES, gestion des clés
- **Interface** : Terminal, expérience utilisateur
- **Architecture** : Multi-threading, design modulaire
